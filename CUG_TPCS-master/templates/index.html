Web 是 CTF 的主要题型，题目涉及到许多常见的 Web 漏洞，如文件包含、代码执行、上传漏洞、SQL 注入等，需要选手进行相关漏洞利用拿下服务器权限。题目环境比较接近真实环境是CTF中的一个重要方向。

Pwn 在黑客俚语中代表着攻破，获取权限，在 CTF 比赛中它代表着溢出类的题目，其中常见类型溢出漏洞有整数溢出、栈溢出、堆溢出等,主要考查参赛选手对漏洞的利用能力。

Reverse 主要涉及到软件逆向、破解技术等，要求有较强的反汇编、反编译功底。主要考查参赛选手的逆向分析能力。

Crypto 考察各种加解密技术，包括古典加密技术、现代加密技术甚至出题者自创加密技术，主要考查参赛选手密码学相关知识点,通常会和其他题目相结合。

Misc 即安全杂项，题目涉及隐写术、流量分析、电子取证、数据分析、大数据统计等，覆盖面比较广，主要考查参赛选手的各种基础综合知识。

Real world 即现实生活真实存在的漏洞，出这类题主要是想让同学们了解真实的漏洞。

OSINT 即开源网络情报(Open source intelligence),是一种情报搜集手段，从各种公开的信息资源中寻找和获取有价值的情报。
